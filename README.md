# Первое задание

## Описание
Микросервисное приложение для управления заказами с использованием Go, Kafka, PostgreSQL и Redis.

## Технологии
- **Backend**: Go
- **Сообщения**: Apache Kafka
- **База данных**: PostgreSQL 17
- **Кэш**: Redis 7
- **Веб-фреймворк**: Gin
- **Контейнеризация**: Docker

## Архитектура
- **app**: Основное Go-приложение (порт 3000)
- **broker**: Kafka брокер (порт 9092)
- **db**: PostgreSQL база данных (порт 5433)
- **redis**: Redis кэш (порт 6379)

### Предварительные требования
- Docker и Docker Compose
- Go 1.24+

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Главная страница |
| GET | `/order/:order_uid` | Получить заказ по UID |
| GET | `/status` | Проверка статуса приложения |

| Команда | Описание |
|---------|----------|
| `make build` | Сборка образов |
| `make up` | Запуск сервисов |
| `make down` | Остановка сервисов |
| `make restart` | Перезапуск |
| `make test` | Запуск тестов |
| `make cover` | Покрытие тестов |
| `make clean` | Полная очистка |

### Установка
1. Клонируйте репозиторий:
```bash
git clone https://github.com/artyomkorchagin/first-task.git
cd first-task
```
2. Запустите приложение:
```bash
make up
```
